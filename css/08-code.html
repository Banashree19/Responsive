Fixed Height

Add the class .pre-scrollable.

pre, code, samp, kbd {
    background-color: #f7f7f9;
    border: 1px solid #e1e1e8;
    color: #dd1144;
    padding: .5em;
}

code {
    padding: .067em .25em;
    white-space: nowrap;
}

pre code,
pre samp {
    border: none;
    padding: 0;
    white-space: pre-wrap;
}

.pre-scrollable {
    max-height: 20em;
    overflow-y: scroll;
}
