CSS

Responsive differs from other CSS frameworks in that it is built specifically not to force opinionated design on the developer. Instead it's been jam-packed with useful functionality to allow you to build a truly cross-browser responsive experience:

A powerful grid system is included with right-to-left language support built in as standard. Forms, buttons, and other elements have also been normalized and enhanced to iron out bugs and make it simple for you to create your award winning site.

Check out the details in each section.
