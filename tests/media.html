External Media

Video containers are scaled when embedding a video from YouTube, Vimeo, or another site that uses iframe, embed, video, or object elements.

Youtube

Add class .media to the media container.

Dailymotion

Add class .media to the media container.

Google Maps

Add classes .media to the media container.


View Larger Map

Vimeo

Add classes .media and .vimeo to the media container.

Vine

Add classes .media and .vine to the media container.

Instagram

Add classes .media and .instagram to the media container.

Adding the class .widescreen sets up the media container for widescreen videos.

Twitter, Facebook, SoundCloud, Storify, and Scribd (select 'Autosize') already provide responsive embed code.
